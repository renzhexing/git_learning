gti 命令

  • 1. git add . 添加所有修改和新增数据到暂存区
  • 2. git add -A 提交所有被删除、被替换、被修改和新增的文件到数据暂存区
  • 3. git add -u 提交所有被删除和修改的文件到数据暂存区
  • 4. git commit -m'[此次提交备注]' 添加到本地版本库
  • 4. git commit -am'[此次提交备注]' 不通过添加到暂存区就直接提交
  • 5. git reset --hard 清空暂存区的修改
  • 6. git log 查看提交记录
  • 6. git log -n4 查看最近4次的提交记录
  • 6. git log --oneline 查看提交记录简洁模式
  • 6. git log --all 查看本地所有分支记录
  • 6. git log --graph 查看分支演进历史
  • 6. git branch -v 查看本地所有分支
  • 6. git mv a b 将a命名为b
  • 6. git cat-file -t 查看类型
  • 6. git checkout -b [name] 新建本地分支
  • 6. git cat-file -t [哈希码] 查看类型
  • 6. git cat-file -p [哈希码] 查看内容
